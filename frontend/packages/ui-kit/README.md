# @resq/ui-kit

Shared design system and component library.

## Features
- **Atomic Components**: Standardized Buttons, Inputs, Cards, and Modals.
- **Theming**: System-aware Dark Mode support via Tailwind.
- **Accessibility**: ARIA-compliant patterns for high-stress scenarios.
- **State Feedback**: Specialized Loading, Error, and Offline banners.
