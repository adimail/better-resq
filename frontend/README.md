# ResQ Frontend (Bun Monorepo)

A workspace-based monorepo containing shared logic and specialized PWAs.

## Structure
- `apps/`: Deployable applications.
- `packages/`: Shared libraries used by the apps.

## Key Technologies
- **TanStack Query**: Server-state management and offline persistence.
- **MapLibre GL**: Hardware-accelerated hazard visualization.
- **Zustand**: Lightweight client-side UI state.
- **Tailwind CSS**: Shared design system.
